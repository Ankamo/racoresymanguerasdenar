# Racores y Mangueras de Nariño

## 1. Project Description
Sitio web corporativo para "Racores y Mangueras de Nariño", empresa dedicada a la comercialización de productos para mantenimiento de vehículos pesados (camiones, maquinaria industrial, transporte de carga). El sitio transmite confianza, solidez y experiencia en el sector automotriz industrial, posicionando la marca como proveedor líder.

**Target users:** Empresas de transporte de carga, constructoras, mineras, industrias agrícolas, talleres mecánicos.
**Core value:** Soluciones confiables con repuestos certificados, lubricantes premium y asesoría técnica especializada.
